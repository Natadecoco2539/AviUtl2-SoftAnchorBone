# AviUtl2-SoftAnchorBone
AviUtl2用の、3点アンカーで髪や服を揺らす軽量ボーン風スクリプト

作成者: [なたのこ/Natadecoco_2539]

■ **概要**
SoftAnchorBoneは、AviUtl2用のスクリプトです。  
3点アンカーを使って、髪の毛・服・スカートなどを揺らします。

■ **特徴**
- 3点アンカーで揺れ範囲を指定
- 髪・服・スカート向けプリセット
- ループ素材化に対応

- ■ **使い方**
1. `SoftAnchorBone.anm2` をAviUtl2のスクリプトフォルダに入れる
2. 揺らしたい画像に適用する
3. アンカーを上から順に配置する

- 1点目：てっぺん
- 2点目：揺れ始める位置
- 3点目：先端
